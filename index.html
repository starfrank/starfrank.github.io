蛙泳学习打卡

发表于 2017-11-08 | 更新于 2017-11-08 18:37:42 | 分类于 杂七杂八
蛙泳学习打卡 此贴用来记录本人学习游泳的过程，直至使用蛙泳的泳姿在泳池可以游一个来回= = ###Day1（7.5）： 上一次不知道在什么时候下的水了。。。 时隔多年再次下水，跑了一天澡，熟悉熟悉水 = = ###Day2 （7.15）： 隔了十天，第二次下水 算是有了突破性进展： 在水下憋气的时间变长了，由最初的5秒能坚持10多秒； 实践了之前看的“鸡蛋漂浮” 蜷缩成一团，双手抱膝，自动浮在水面； 简单的能漂浮几秒，身体伸直成一条直线； 简单学了点蛙泳中腿的动作， 核心：脚往内扣，用脚掌压水，整个动作为，大腿并拢，外八收腿，靠脚掌往外压水，伸直并拢双腿（前进的主要动力来源），收腿，循环上述流程； 手部前伸的部分尽可能双手手掌合起来与水面垂直，感觉阻力更小； 沉迷上了从泳池边缘猛蹬一下后窜出去划几米，很爽 终于找到点游泳的感觉 ###Day3 （7.17）： 水里泡了一小时，一直在找漂浮的感觉，有点进步，能漂个十来秒 = = 关键点！！！水下必须憋气，减少吐气，吐气会导致迅速下沉！！！ 水下憋住气，少吐气，等上半身浮起来，轻点地，两腿尽量靠近绷直，防止分开。双手前伸，尽量往身体后方伸，胸会往水下压！！！！自动就浮起来了！！ 背靠墙壁，手往头上伸，尽量拉直身体，靠近墙壁，找感觉！！ 闲的时候尝试吐气，触摸池底，有点意思！！ ###Day4 （7.18）： 继续游泳搞起！！终于能不靠边，直接成功漂起来了！！！！大突破！！！！yes！！！！！ 漂的感觉就是全身拉直，想想靠墙的感觉，收腹挺胸！！水里少吐气，尽量憋气！ 漂起来的基础上，腿夹住浮板，手扶岸边，全身漂起来，尝试换气！！不是很流畅，经常喝水。。。。= = 初步练习换气：头进入水下，鼻子出气，出水面的时候，迅速喊出“bo or pa”，把嘴里的气吐干净，再迅速吸一口气，进入水下，鼻子呼气，重复过程15次。。。感觉贼鸡儿不流畅。。。到后面鼻子会有水，还待联系 练习蛙泳腿：漂浮的基础上，不换气，憋气，蛙泳腿游，憋不住的时候停下站立出水换气，换完继续憋气游！5-6口气游20多米，终于有了点游泳的感觉，也离开了岸边！！很开心！！！每天来，集中练，效果不错！ ###Day5 （7.19）： 连续第3天游泳，主要练习了换气！ 站在水中换气，连续换15次，练了一段时间； 大腿夹住浮板，手扶岸边，连续抬头换气，练了一段时间，换气效果较差； 蛙泳腿+换气，手扶浮板，尝试换气，换两次后沉下去了。。。。 = = sad！！！ 感觉还是抬头幅度太大的原因么。。。坚持时间不长 最后尝试了下不用浮板，脚蹬岸边，滑行一段距离后，尝试蹬腿后+手部运动+换气，动作幅度很大，也是换两口气后就沉下去了 = = 关键的换气，还待联系！and 连续3天游泳后大腿略疼，适度休息后继续搞起！ 蛙泳的一个连贯动作和技巧：划手抱水、抬头吸气、收手后收腿（把手收至胸前，小腿收起，即换气的时候腿是收起来的）——伸手埋头呼气、猛蹬腿、夹水——身体平直、漂一会儿 ###Day6 （7.21）： 距上次游泳后隔了一天，再次进入水池 0 0 主要还是练习换气，进展不是很顺利，心塞。。。 = = 手扶浮板+蛙泳腿，勉强换三次气，比之前倒是强了点。有了点感觉，身体较平，换气时仅头部抬头低头，肩膀不动，沉的慢点，能多换几次。 直接蛙泳腿+蛙泳手，尝试换气，稍微强了点，蹬2-3次腿，换一次气，好的时候能换个3至4次气。。 目前主要问题： 出水时间太短，换气不充分，还没吐干净，勉强吸了一口气，已经再次进入水中 进入水中沉的太快，直接沉入靠近水底的位置，需要蹬两次腿才能上来 动作不流畅，手部动作和腿部动作衔接有问题 腿部动作不规范，前进动力不足 问题很多，还需进一步学习相关姿势 = = 迅速夹臂，能提高出水的时间 虽然学习遇到了瓶颈，水性还是慢慢有所加强，算是好事~~总有学会的时候！！！ 网上看到一篇关于蛙泳的好文： 貌似主要是波蛙，与平时的平蛙还有区别，仅供参考蛙泳好文 简单注意的几点： 滑行时，眼睛看池底，换气时抬头幅度不要过大，稍微抬头，眼睛往前看一两英尺的地方，保持头部、颈部和整个脊椎呈一条直线； 外划，内划阶段，快速锁肘，耸肩，出水换气，换完立即前冲伸臂； 滑行阶段，用力下压胸部，提高髋部，立身吸气时，臀部向前下方下压，往前顶，收腿，后脚跟上抬至臀部，只收膝盖，髋部尽量不动！ 收脚时，脚后跟越靠近臀部，蹬腿越有力，收脚速度越快，游得速度越快，蹬腿过程中一定要保持勾脚，要用脚面蹬水蹬实 ###Day7 （7.25）： 歇了4天之后又一次下水，主要还是学习蛙泳换气 = = 比上一次坚持的时间长点了，能换个3至4次气，关键：换气不足时，进入水下，脚部动作规范后，沉入水中后，坚持流线型漂的动作，蹬两次水整 ...

羽毛球技术要点

发表于 2017-03-28 | 更新于 2017-03-28 18:37:42 | 分类于 杂七杂八
羽毛球技术要点 断断续续打羽毛球也有个两年多了，一直处于瞎比打状态 = = 最近实验室高手越来越多，怕是要被虐死，赶紧看两集教学视频压压惊，此贴专门记录观看教学视频的总结，希望总结完打球水平能更进一步！！ 发球接发球 正手高远球： 击球时，手臂内旋，手腕要有展腕的动作，不能维持屈腕动作； 上身自然转体； 正手平高球： 与高远球基本相同，主要是击球部分手臂带动手腕，球高度不高也不低； 击球拍面小于45度； 正手平射球： 击球点要比之前高，引拍动作要小； 击球要迅速，拍面仰角要小； 正手发网前小球： 手臂不内旋，握拍要送，手腕收腕轻推发力； 反手平高球： 手肘为轴，小臂内旋，由后向前作回环半弧形挥动； 击球瞬间，拇指上顶发力； 反手平射球： 击球点尽可能高，拍面与地面尽可能90度，迅速向前击球； 反手发网前小球： 收腕动作与反手平高球一样； 手腕由外展至内收，捻动发力，主要靠手腕和手指； 斜拍面向前切球击球； 接球： 脚后跟不着地，重心在前脚掌； 左脚在前半步（对我而言的） 网前步伐： 一步，右脚蹬地，左脚往前一步； 两步，右脚往前蹬一步，左脚前跨一步； 击球 正手高远球： 前臂急速外旋，带动手腕加速向前上方推动，正拍面将球击出，击球点在手臂伸直最高侧； 控制拍面，实现不同角度的击球； 正手后场吊球： 击球点比高远球要前，要有向下的力； 切击球，切斜线的球切击动作幅度要大； 正手后场杀球： 击球点在身体前侧； 调动全身的力量，前下快速压球，还是正拍面，击球爆发力大； 正手劈球： 斜线切击球； 反手后场高远球： 反手握拍，转体引拍，含胸，手臂内旋，由前下方向上方挥动； 挺身，大拇指顶力，正拍面击球； 击球后手臂制动； 正手网前搓放球： 击球点比肩高，拍面为斜的； 搓球（进攻），靠摩擦力将球挑过网，不要向上发力； 小臂外旋，半弧形引拍； 搓球分两种：由外向内（位置较好），由内向外（斜线来球）； 放小球（防守）：球拍低于球托，轻轻抬击球，放网前小球； 反手网前搓放小球： 小臂内旋半弧形推动； 离网前较近，外旋； 正反手网前推球： 以比较高的击球点，比较平的角度推出去； 手肘为轴，手臂外旋至内旋，手腕伸腕至展腕； 食指发力，推球出去； 正反手网前勾球： 击球点高，网前切击； 击球点低，侧方送球；
