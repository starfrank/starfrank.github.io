羽毛球技术要点

断断续续打羽毛球也有个两年多了，一直处于瞎比打状态 = =

最近实验室高手越来越多，怕是要被虐死，赶紧看两集教学视频压压惊，此贴专门记录观看教学视频的总结，希望总结完打球水平能更进一步！！

发球接发球

正手高远球：

  • 击球时，手臂内旋，手腕要有展腕的动作，不能维持屈腕动作；
  • 上身自然转体；

正手平高球：

  • 与高远球基本相同，主要是击球部分手臂带动手腕，球高度不高也不低；
  • 击球拍面小于45度；

正手平射球：

  • 击球点要比之前高，引拍动作要小；
  • 击球要迅速，拍面仰角要小；

正手发网前小球：

  • 手臂不内旋，握拍要送，手腕收腕轻推发力；

反手平高球：

  • 手肘为轴，小臂内旋，由后向前作回环半弧形挥动；
  • 击球瞬间，拇指上顶发力；

反手平射球：

  • 击球点尽可能高，拍面与地面尽可能90度，迅速向前击球；

反手发网前小球：

  • 收腕动作与反手平高球一样；
  • 手腕由外展至内收，捻动发力，主要靠手腕和手指；
  • 斜拍面向前切球击球；

接球：

  • 脚后跟不着地，重心在前脚掌；
  • 左脚在前半步（对我而言的）

网前步伐：

  • 一步，右脚蹬地，左脚往前一步；
  • 两步，右脚往前蹬一步，左脚前跨一步；

击球

正手高远球：

  • 前臂急速外旋，带动手腕加速向前上方推动，正拍面将球击出，击球点在手臂伸直最高侧；
  • 控制拍面，实现不同角度的击球；

正手后场吊球：

  • 击球点比高远球要前，要有向下的力；
  • 切击球，切斜线的球切击动作幅度要大；

正手后场杀球：

  • 击球点在身体前侧；
  • 调动全身的力量，前下快速压球，还是正拍面，击球爆发力大；

正手劈球：

  • 斜线切击球；

反手后场高远球：

  • 反手握拍，转体引拍，含胸，手臂内旋，由前下方向上方挥动；
  • 挺身，大拇指顶力，正拍面击球；
  • 击球后手臂制动；

正手网前搓放球：

  • 击球点比肩高，拍面为斜的；
  • 搓球（进攻），靠摩擦力将球挑过网，不要向上发力；
  • 小臂外旋，半弧形引拍；

  • 搓球分两种：由外向内（位置较好），由内向外（斜线来球）；

  • 放小球（防守）：球拍低于球托，轻轻抬击球，放网前小球；

反手网前搓放小球：

  • 小臂内旋半弧形推动；
  • 离网前较近，外旋；

正反手网前推球：

  • 以比较高的击球点，比较平的角度推出去；
  • 手肘为轴，手臂外旋至内旋，手腕伸腕至展腕；
  • 食指发力，推球出去；

正反手网前勾球：

  • 击球点高，网前切击；
  • 击球点低，侧方送球；
