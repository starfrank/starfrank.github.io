蛙泳学习打卡

此贴用来记录本人学习游泳的过程，直至使用蛙泳的泳姿在泳池可以游一个来回= =

###Day1（7.5）：

上一次不知道在什么时候下的水了。。。

时隔多年再次下水，跑了一天澡，熟悉熟悉水 = =

###Day2 （7.15）：

隔了十天，第二次下水

算是有了突破性进展：

  • 在水下憋气的时间变长了，由最初的5秒能坚持10多秒；

  • 实践了之前看的“鸡蛋漂浮”

    蜷缩成一团，双手抱膝，自动浮在水面；

  • 简单的能漂浮几秒，身体伸直成一条直线；

  • 简单学了点蛙泳中腿的动作，

    核心：脚往内扣，用脚掌压水，整个动作为，大腿并拢，外八收腿，靠脚掌往外压水，伸直并拢双腿（前进的主要动力来源），收腿，循环上述流程；

  • 手部前伸的部分尽可能双手手掌合起来与水面垂直，感觉阻力更小；

  • 沉迷上了从泳池边缘猛蹬一下后窜出去划几米，很爽

终于找到点游泳的感觉

###Day3 （7.17）：

水里泡了一小时，一直在找漂浮的感觉，有点进步，能漂个十来秒 = =

关键点！！！水下必须憋气，减少吐气，吐气会导致迅速下沉！！！

水下憋住气，少吐气，等上半身浮起来，轻点地，两腿尽量靠近绷直，防止分开。双手前伸，尽量往身体后方伸，胸会往水下压！！！！自动就浮起来了！！

背靠墙壁，手往头上伸，尽量拉直身体，靠近墙壁，找感觉！！

闲的时候尝试吐气，触摸池底，有点意思！！

###Day4 （7.18）：

继续游泳搞起！！终于能不靠边，直接成功漂起来了！！！！大突破！！！！yes！！！！！

漂的感觉就是全身拉直，想想靠墙的感觉，收腹挺胸！！水里少吐气，尽量憋气！

漂起来的基础上，腿夹住浮板，手扶岸边，全身漂起来，尝试换气！！不是很流畅，经常喝水。。。。= =

初步练习换气：
头进入水下，鼻子出气，出水面的时候，迅速喊出“bo or pa”，把嘴里的气吐干净，再迅速吸一口气，进入水下，鼻子呼气，重复过程15次。。。感觉贼鸡儿不流畅。。。到后面鼻子会有水，还待联系

练习蛙泳腿：
漂浮的基础上，不换气，憋气，蛙泳腿游，憋不住的时候停下站立出水换气，换完继续憋气游！5-6口气游20多米，终于有了点游泳的感觉，也离开了岸边！！很开心！！！
每天来，集中练，效果不错！

###Day5 （7.19）：

连续第3天游泳，主要练习了换气！

站在水中换气，连续换15次，练了一段时间；

大腿夹住浮板，手扶岸边，连续抬头换气，练了一段时间，换气效果较差；

蛙泳腿+换气，手扶浮板，尝试换气，换两次后沉下去了。。。。 = = sad！！！

感觉还是抬头幅度太大的原因么。。。坚持时间不长

最后尝试了下不用浮板，脚蹬岸边，滑行一段距离后，尝试蹬腿后+手部运动+换气，动作幅度很大，也是换两口气后就沉下去了 = =

关键的换气，还待联系！and 连续3天游泳后大腿略疼，适度休息后继续搞起！

蛙泳的一个连贯动作和技巧：划手抱水、抬头吸气、收手后收腿（把手收至胸前，小腿收起，即换气的时候腿是收起来的）——伸手埋头呼气、猛蹬腿、夹水——身体平直、漂一会儿

###Day6 （7.21）：

距上次游泳后隔了一天，再次进入水池 0 0

主要还是练习换气，进展不是很顺利，心塞。。。 = =

手扶浮板+蛙泳腿，勉强换三次气，比之前倒是强了点。有了点感觉，身体较平，换气时仅头部抬头低头，肩膀不动，沉的慢点，能多换几次。

直接蛙泳腿+蛙泳手，尝试换气，稍微强了点，蹬2-3次腿，换一次气，好的时候能换个3至4次气。。

目前主要问题：

  • 出水时间太短，换气不充分，还没吐干净，勉强吸了一口气，已经再次进入水中
  • 进入水中沉的太快，直接沉入靠近水底的位置，需要蹬两次腿才能上来
  • 动作不流畅，手部动作和腿部动作衔接有问题
  • 腿部动作不规范，前进动力不足

问题很多，还需进一步学习相关姿势 = =

迅速夹臂，能提高出水的时间

虽然学习遇到了瓶颈，水性还是慢慢有所加强，算是好事~~总有学会的时候！！！

网上看到一篇关于蛙泳的好文：

貌似主要是波蛙，与平时的平蛙还有区别，仅供参考
蛙泳好文

简单注意的几点：

  • 滑行时，眼睛看池底，换气时抬头幅度不要过大，稍微抬头，眼睛往前看一两英尺的地方，保持头部、颈部和整个脊椎呈一条直线；
  • 外划，内划阶段，快速锁肘，耸肩，出水换气，换完立即前冲伸臂；
  • 滑行阶段，用力下压胸部，提高髋部，立身吸气时，臀部向前下方下压，往前顶，收腿，后脚跟上抬至臀部，只收膝盖，髋部尽量不动！
  • 收脚时，脚后跟越靠近臀部，蹬腿越有力，收脚速度越快，游得速度越快，蹬腿过程中一定要保持勾脚，要用脚面蹬水蹬实

###Day7 （7.25）：

歇了4天之后又一次下水，主要还是学习蛙泳换气 = =

比上一次坚持的时间长点了，能换个3至4次气，关键：换气不足时，进入水下，脚部动作规范后，沉入水中后，坚持流线型漂的动作，蹬两次水整个身体又重新漂浮起来，感觉接近水面后继续尝试换气。

换气部分，出水后尽量张大嘴，“po”的音尽量发出来，尽量迅速的吐气吸气，目前换气还是不够快，吸气部分不足 = =

3蹬腿1换气还没学会，前路漫漫 = =

###Day8 （7.27）：

继续练习蛙泳、换气的结合，总共就游了一个小时，被抓回来打麻将。。。= =

发现一个论坛：游泳梦工厂，有的帖子还不错，就是网站服务器感觉不稳定，连接时断时续，会存在登不上的情况

游泳梦工厂

暂时没什么感觉，先写这么多

###Day9 （8.7）：

高温假出去浪了一周，回来继续学。。。先找了找感觉

看了看论坛，主要是在之前的基础上加入了划手，即蹬一次腿，划一次手，两次或三次换一次气，还是停留在换两口的状态；

加入划手后，滑行距离长了很多，手上的动力也很大，游一个小时明显胳膊很累

划手过程中，划水部分两手一定要向外向下，角度大一点，出水时间长一点，动力足一点

抱水后的夹肘尽量快，能减小整个收手的阻力，效果不错。

###Day10 （8.8）：

还是练习换气，在水中要主动吐气，尽量减少气；

在抬头吸气时尽可能加一些腹部力量进行吸气，增强吸气的内容；

在水中动作不要着急，稍慢一些，蹬完腿漂一会再划水换气，划水也不着急；

###Day10 （8.9）：

继续换气，今天较之前有较大进展，在换气的过程中终于能吸到气了，状态好的话能换4-5口气，算是突破性进展！！ 0_0

主要的要点：

  • 换气中的吸气和呼气部分一定要用喉咙发力！！！

换气的时候，不是口腔吹气，要从咽喉深处吐出！！！

吸气的时候要喉咙吸气，深吸到咽喉，不能只吸到口腔！否则会感觉空气没进到肺里！！！

  • 换气出水前漂浮状态稍微主动地吐一些气，方便后面的吸气
  • 划手过程中的划水部分可以适当加大角度，而且一定要加上夹肘的动作，出水的时间会有明显加长！！！

存在的问题：

目前每次换完气继续伸手低头后，下沉较多，脚蹬水的效果不好，整体上浮不大，需要借用手臂力量，整体游下来很累，需要规范脚步动作，提高蹬水效率！！！
